# LangChain Gemini Tool-Using Chatbot

This project is a **tool-using conversational AI chatbot** built using:

- **LangChain**
- **Google Gemini API**
- **Custom Python tools**
- **Manual agent loop with memory**
- **Secure API key handling using `config.py`**
- **Virtual Environment for dependency isolation**

---

## Features

-  Gemini-powered responses
-  Custom tools (e.g., multiplication, discount calculator)
-  Automatic tool invocation by the agent
-  Conversation memory
-  Enforced JSON output format
-  Secure API key (not exposed to GitHub)
-  Virtual environment support
